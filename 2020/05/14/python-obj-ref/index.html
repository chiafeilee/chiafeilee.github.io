==运算符比较两个对象的值（对象中保存的值），is比较对象的标识（内存地址）
在变量和单例值(singleton)之间的比较时，应该使用is，例如None

为什么用is比较None

什么是None？

The sole value of the type NoneType. None is frequently used to represent the absence of a value, as when default arguments are not passed to a function

Facts

  • ==运算符是语法糖，等同于对象的x.__eq__(y)——继承自object对象，这个方法可被override
  • 由上一条得is运算符比==运算符要快，因为不用寻找特殊方法__eq__()，is直接比较ID
  • 关键字None是一个对象
  • None是一个singleton值——NoneType类型的单例值

对象复制

Python中赋值表达式不复制对象，仅仅是创建一个链接到目标对象，即共享源对象的引用

浅复制(Shallow Copy)

复制最外层的容器，副本中元素(child objects)是源容器中元素的引用，构造方法及[:]或copy包中的copy方法做浅复制

深复制(Deep Copy)

副本不共享内部对象引用

浅复制和深复制的不同点仅仅在于复合对象(compund objects)上——对象包含其他对象，由此可得出一个简单的结论，即浅复制是一个one level deep

变量

Important Facts
Python不像其他语言如C可以声明一个变量而不赋值，必须在赋值表达式中声明并给定初始值，如None

概念：命名(Naming)和绑定(Binding)

Python中的“变量”和其他的语言如C、Go中的概念很不一样——先在内存中申请个“盒子”（定义变量），然后往里装东西（赋值），Python是先在内存里未命名的“盒子”里装东西，然后贴上“标签”（变量名），Python中“变量”通常称为“名称”（names）

Names refer to objects. Names are introduced by name binding operations
—— Python Execution Model

名称在绑定操作的时候引入，即通常意义上的“赋值”操作，有4种情形会发生绑定操作：

  • 传给函数的形式参数
  • import语句
  • 类与函数定义
  • 以标识符为目标的赋值，如for循环的开头、with语句、except子句的as之后
