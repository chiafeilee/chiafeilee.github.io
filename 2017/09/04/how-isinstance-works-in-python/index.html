实现原理

Python的底层是用C语言实现的，Python中一切都是对象，那么相应的在CPython中PyObject就是Python对象的基石，它的定义如下：
Include/object.h

在宏PyObject_HEAD中定义初始化PyObject各个部分的值，其中包括了PyTypeObject这个用来描述类型对象的结构体，它就相当于一个情报局，记录了类型对象的方方面面，即包含了各种元数据包括分配内存的大小、类型名等等，Python的类型系统就是基于此对象构建的，例如int、string等。知道了这些就可以大概的猜测下isinstance的大概的处理流程了：首先，判断对象或实例的类型与所比较的对象的类型是否一致，
